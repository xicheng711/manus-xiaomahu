# Trend Chart Reference Design Notes

Key elements from the reference image:
1. **Week/Month toggle** - pill-shaped segmented control at top (周/月)
2. **Date range navigation** - "本周 2019年6月10至16日" with left/right arrows to navigate weeks
3. **Mood Index card** - thermometer-style gauge on left, large number (6.8) on right, with comparison stats (相比当月: +57%, 相比上周: -26%)
4. **Mood Distribution** - emoji icons with count badges, showing distribution of moods (迷惘x3, 充实x3, 得意x1, 低落x2), total count and most frequent mood
5. **Emotion Change curve** - smooth area/line chart showing mood changes over the week, with emoji Y-axis labels (happy/neutral/sad faces)
6. **Clean card-based layout** - each section in its own white card with rounded corners
7. **Soft color palette** - light teal/cyan accent colors
